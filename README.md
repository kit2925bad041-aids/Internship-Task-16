# 🎓 Student Performance Analysis

## 📌 Overview
This project performs comprehensive analysis of student academic performance using Python. It calculates total marks, average scores, grades, identifies top performers, and generates insightful visualizations for educational analytics.

## 🚀 Features
- Student dataset analysis
- Total and average score calculation
- Automatic grade assignment
- Top performer identification
- Statistical summary generation
- Subject-wise performance analysis
- Grade distribution analysis
- Data visualization with charts
- CSV report generation

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
